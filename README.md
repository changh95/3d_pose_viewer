# 3d_pose_viewer
A simple 3D coordinate viewer. Currently supports KITTI dataset pose format.
Inspired by GaoXiang's slambook


## Demo

![](./3d_pose_viewer.gif)

## How to build

Pre-requisite: python3

```
./build_dependencies.py
````

## How to run

```
./build/pose_viewer ./cam0_to_world.txt
```
